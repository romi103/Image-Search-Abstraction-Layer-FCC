API Basejump: Image Search Abstraction Layer

User stories:
    1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.
    3) I can get a list of the most recently submitted search strings.

Example usage:

Search

https://image-search-abstraction-fcc.herokuapp.com/search/london?offset=5

Will return response in JSON file where search/london part contains serached phrease and ?offset=5 will specify paginaton.

Retrieving latest searches

https://image-search-abstraction-fcc.herokuapp.com/latest

Will return a list of ten most recently submitted search strings.
